# 🎮 Dota 2 Match Analyzer

Скрипт для анализа ваших матчей через OpenDota API.

## Установка

```bash
cd .../dota-analyzer
pip install -r requirements.txt
```

## Запуск

```bash
python match_analyzer.py
```

## Как узнать свой Steam ID

1. Зайдите на https://www.opendota.com/
2. Введите свой никнейм в поиске
3. Скопируйте ID из URL (цифры после `/players/`)

Или используйте https://steamid.io/ для конвертации профиля

## Что анализирует скрипт

- 📊 **Общая статистика**: винрейт, K/D/A, GPM, XPM
- 🦸 **Лучшие герои**: по винрейту (мин. 2 игры)
- ⚠️ **Худшие герои**: для пересмотра пула
- 📍 **Позиции**: эффективность на каждой линии
- 💡 **Рекомендации**: персональные советы
